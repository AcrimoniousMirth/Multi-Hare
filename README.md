<p align="center">
Universal Automated Filament Changer / MMU driver for Klipper (Multi-Toolhead Fork)
</p>

Multi-Hare is a modified fork of Happy-Hare, the original open-source filament changer controller for multi-color printing. Its intention is to add multiple toolhead functionality to Happy-Hare. Happy-Hare's philosophy is to provide a universal control system that adapts to your choice of MMU (Multi-Material Unit). Happy-Hare supports many MMU designs, but while keeping as much as possible of the original code unchanged,Multi-Hare cannot guarantee this.

Multi-Hare is under active development and relies heavily on commits from the original Happy-Hare project. 

This is a personal project not expected to have any reach but if you do use it and find it of value and wish to contribute, please do so to the original creator of Happy-Hare here: https://www.paypal.me/moggieuk.
